# SLT-Узел Тольятти — статический сайт

Полимерные трубы и фитинги СЛТ Аква (SLT BLOCKFIRE и др.). Сайт — чистый HTML/CSS/JS.

## Обновить сайт на GitHub Pages

```bash
git add -A
git commit -m "Кратко, что изменили"
git push
```

Ветка по умолчанию: **main**. После `push` деплой идёт автоматически (см. `.github/workflows/pages.yml`).

Подробно: файл **HOSTING.txt** (Netlify, GitHub, проверки).

## Локальный просмотр

Откройте `index.html` в браузере или используйте Live Server в редакторе.
